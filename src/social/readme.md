# Социальные кнопки

Инлайновый тип. [Пример](http://sedona.stage.constlab.ru/blocks/social/)

## Особенности

* Может располагаться внутри любого типа блока
* По-умолчанию каждая соцсеть имеет брендовые цвета. Может принудительно иметь светлый и темный тон

@social-darken-hover-background-color: lighten(@social-darken-background-color, 10%);
