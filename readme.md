# Набор компонентов

## Принципы

* Цвета блоков зависят от тона всего сайта
* Сетка строится от размера шрифта
* Верхний элемент отталкивает нижний

## Лейаут

Рекомендуемая разметка тела документа.

```html
<div class="page">
  <header class="header">
    Шапка
  </header>

  <main class="main">
    Контент
  </main>

  <aside class="aside">
    Сайдбар
  </aside>

  <footer class="footer">
    Подвал
  </footer>
</div>
```
