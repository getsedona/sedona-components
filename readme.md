# Sedona components

Общие компоненты для проектов компании «Конст Лаб».

## Установка

```
npm i sedona-components
```

## Принципы

### Разметка тела документа

```html
<div class="page">
  <header class="header">
    Шапка
  </header>

  <main class="main">
    Контент
  </main>

  <aside class="aside">
    Сайдбар
  </aside>

  <footer class="footer">
    Подвал
  </footer>
</div>
```
